Adversarial Detection in IDS System using Advanced Machine Learning

This project focuses on enhancing the Intrusion Detection System (IDS) by detecting adversarial attacks using advanced machine learning techniques, such as Convolutional Neural Networks (CNN), Recurrent Neural Networks (RNN), and Generative Adversarial Networks (GAN). The project also explores the application of SMOTE for handling class imbalance in the NSL-KDD dataset.

Key Features:

  • Data Preprocessing and Augmentation using SMOTE
  • Model training with CNN, RNN, and GAN
  • Evaluation of model performance against adversarial attacks (FGSM and GAN)
  • Comparison of model robustness and accuracy

Technologies Used: Python, TensorFlow, Keras, Scikit-learn, NSL-KDD Dataset
